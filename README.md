# Nexus Fashion

Clean rebuilt storefront project created from the working `Nexus website frontend` code, without the mixed `file 1` pages.

## Run locally

From this folder:

```powershell
py -3 server.py
```

Then open:

```text
http://127.0.0.1:8000
```

## Main files

- `site-app.js`: app runtime and page rendering
- `catalog.js`: product/catalog data
- `styles.css`: shared design system and layout
- `resources/`: images and brand assets
- `supabase-schema.sql`: optional backend schema
